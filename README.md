# Smart Expense Tracker

A responsive expense tracking web application built with HTML, CSS, and JavaScript that allows users to manage expenses through authentication, form validation, and an interactive dashboard.

## Features
- User login and sign-up flow
- Form validation for authentication
- Add and delete expenses
- Interactive dashboard
- Responsive UI for different screen sizes

## Tech Stack
- HTML
- CSS
- JavaScript
